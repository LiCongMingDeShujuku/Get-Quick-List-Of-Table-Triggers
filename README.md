## 中文
这里有一些可以显示特定表中所有触发器的列表的快速sql逻辑。我使用msdb..sysjobhistory表作为示例。


## English
Here’s some quick sql logic to show you a list of all the triggers in a particular table. I’m using the msdb..sysjobhistory table as an example.

---
## Logic
```SQL
select
'table_name'    = stable.name
,   'trigger_name'  = strig.name
from
sys.triggers strig join sys.tables stable on strig.parent_id = stable.object_id where
stable.name = 'sysjobhistory'
order by
stable.name, strig.name asc


```
